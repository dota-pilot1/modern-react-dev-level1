# 로그인 페이지 레이아웃 구조

이 프로젝트의 로그인 페이지 레이아웃은 다음과 같이 구성되어 있습니다.

## 1. 컴포넌트 구조

- `src/components/LoginLayout.tsx`: 로그인 전용 레이아웃 컴포넌트
- `src/pages/Login.tsx`: 로그인 폼 및 UI

## 2. 레이아웃 설명

### LoginLayout.tsx
- 전체 화면을 2개의 컬럼(grid)으로 분할
  - **왼쪽(브랜드/설명 영역)**: 데스크탑에서만 보임(`lg:grid-cols-2`, `hidden lg:flex`)
  - **오른쪽(로그인 카드 영역)**: 로그인 폼이 들어가는 카드(`max-w-[480px]`)
- Outlet을 통해 실제 로그인 폼이 렌더링됨
- Tailwind CSS로 반응형 및 스타일 적용

### Login.tsx
- 로그인 폼은 카드 내부에 렌더링됨
- 인풋, 버튼 등은 shadcn/ui 컴포넌트 사용
- 반응형 레이아웃(`w-full`, `max-w-full`, `min-w-0` 등) 적용
- "또는 계속" 구역은 flex로 선과 텍스트를 배치

## 3. 라우팅 구조
- App.tsx 또는 라우트 설정에서 `/login` 경로에 `LoginLayout`이 적용됨
- `LoginLayout` 내부의 `<Outlet />`에 `Login.tsx`가 렌더링됨

## 4. 커스텀 및 확장
- 필요에 따라 카드 너비, 컬럼 수, 배경 스타일 등 커스텀 가능
- 다른 인증 페이지(회원가입 등)도 동일한 레이아웃 구조를 재사용 가능
